# Assets Directory

This directory should contain the following image files for your app:

## Required Images

1. **icon.png** (1024x1024px)
   - App icon for iOS and general use
   - Should be a square image with your app logo
   - Format: PNG with transparency

2. **adaptive-icon.png** (1024x1024px)
   - Android adaptive icon foreground
   - Should be centered and work well on various backgrounds
   - Format: PNG with transparency

3. **splash.png** (1242x2436px recommended)
   - Splash screen image
   - Should contain your app logo/name
   - Format: PNG

4. **favicon.png** (48x48px)
   - Web favicon
   - Format: PNG

## Design Guidelines

- Use high-resolution images (at least 2x the required size for retina displays)
- Keep file sizes reasonable (optimize images)
- Use consistent branding across all assets
- Ensure icons work well on both light and dark backgrounds

## Tools for Creating Assets

- **Figma** (free): https://www.figma.com
- **Canva** (free): https://www.canva.com
- **GIMP** (free): https://www.gimp.org
- **Adobe Photoshop** (paid)

## Quick Creation Tips

1. Create a 1024x1024px square design for the icon
2. Export as PNG with transparency
3. For splash screen, create a design that works in portrait orientation
4. Test your icons on different backgrounds to ensure visibility
